##Â Installation

First, install yo:

```bash
$ npm install -g yo
```

Then, install the generator:

```bash
$ npm install -g @gnodi/generator-lib
```

## Use

### Generate a lib boilerplate

Generate a lib boilerplate using the command:

```bash
$ yo @gnodi/lib
```

Or simply use the Yeoman interface and chose the generator `@gnodi/lib`:

```bash
$ yo
```
